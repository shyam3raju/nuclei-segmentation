📋 STEP-BY-STEP MANUAL GUIDE:

   1. Double-click: 0_INSTALL.bat
      → Installs all Python packages (5-10 min)

   2. Double-click: 1_verify_setup.bat
      → Checks if everything is working

   3. DOWNLOAD DATASET (Choose one option):
   
      Option A: Double-click: 2_download_dataset.bat
         But first you need:
         - Kaggle account (free, sign up at kaggle.com)

         - Put kaggle.json in: C:\Users\YOUR_NAME\.kaggle\
      
      Option B: Manual download:
         - Go to: https://www.kaggle.com/c/data-science-bowl-2018/data
         - Download "data-science-bowl-2018.zip"
         - Extract it
         - Copy the "stage1_train" folder to:
           F:\Opencv\nuclei-segmentation-unet\data\

   4. Double-click: 3_quick_test.bat
      → Quick 5-minute test to verify everything works

   5. Double-click: 4_full_training.bat
      → Full training (30-60 min with GPU, 2-3 hours without)

   6. Double-click: 5_evaluate.bat
      → See how good your model is (Dice score, IoU score)

   7. Double-click: 6_visualize.bat
      → Creates pretty images showing predictions

🎯 WHAT IS THIS PROJECT?

   It's a deep learning model (U-Net) that automatically finds and 
   outlines cell nuclei in microscopy images.
   
   Used in medical research, drug discovery, and disease diagnosis.


❓ HELP FILES:

   - HOW_TO_RUN.txt - Detailed instructions
   - README.md - Full project documentation


🔧 IF SOMETHING GOES WRONG:

   Problem: "pip is not recognized"
   → You need Python installed. Download from python.org

   Problem: "CUDA out of memory"
   → Edit the .bat file, change --batch_size 8 to --batch_size 4

   Problem: "Dataset not found"
   → Make sure you downloaded the data and put it in data\stage1_train\

   Problem: "Module not found"
   → Run 0_INSTALL.bat again


💡 QUICK COMMANDS (if you prefer command line):

   Open PowerShell/CMD in this folder, then:

   pip install -r requirements.txt               (install)
   cd src && python test_setup.py               (verify)
   cd data && python download_data.py           (download data)
   cd src && python train.py --num_epochs 30   (train)
   python evaluate.py                           (evaluate)
   python inference.py --num_samples 10         (visualize)


⏱️ TIME ESTIMATES:

   Installation: 5-10 minutes
   Dataset download: 10-15 minutes
   Quick test: 5 minutes
   Full training: 30-60 min (GPU) or 2-3 hours (CPU)
   Evaluation: 2 minutes
   Visualization: 3 minutes


✅ EXPECTED RESULTS:

   After training, you should get:
   - Dice Coefficient: 0.75-0.85 (higher is better)
   - IoU Score: 0.65-0.75
   
   This means your model is doing a good job at finding nuclei!
